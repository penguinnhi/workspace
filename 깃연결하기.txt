2. github desktop 프로그램 설치
3. github desktop에서 이메일과 이름 확인
4. 3번(create new repository....) 메뉴 선택
5. 폴더를 생성할 위치 결정 + 폴더 생성

8. name은 변경하지말고 그대로 생성!
9. commit- > push
